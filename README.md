# Mcoffee
*A demo app of Gank and Zhihu Daily based on MVP,RxJava,Dagger,Okhttp3,Retrofit2,Glide*

>基于MVP,RxJava,Dagger,Okhttp3,Retrofit2等的demo

>本应用为个人实践所用项目，目的在于学习框架，android design等知识
